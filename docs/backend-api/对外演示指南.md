## 5. 系统架构

```
                        公网用户
                           │
                  https://fun-md.com  (Cloudflare 隧道)
                           │
            ┌──────────────┴──────────────┐
       /api/* 直连                    其余 → Vite
            │                        :3335 (前端 SPA)
            ▼
   ┌─────────────────┐    ┌──────────────────────┐
   │ FastAPI 主后端   │    │ label-system 微服务群 │
   │ :9090           │    │ :8100-8106            │
   │ 推理/计费/GPU/  │    │ IAM/工作流/数据/标注/  │
   │ 工厂/调度/审计   │    │ 智能/模型工厂/网关     │
   └────────┬────────┘    └──────────────────────┘
            │
   ┌────────┴─────────┐
   │ vLLM :8001       │  Qwen3-8B 文本推理（流式）
   │ PostgreSQL :5433 │  业务数据库
   └──────────────────┘
```

### 技术栈
- **前端**：React 19 + antd v6 + React Router v7 + TanStack Query + Vite 7 + Tailwind
- **后端**：FastAPI + SQLAlchemy + Pydantic
- **模型推理**：vLLM（Qwen3-8B，支持流式 SSE）
- **数据库**：PostgreSQL 5433
- **外部模型**：智谱 GLM-4 / 阿里 Qwen（双模型分析）

| 项目 | 状态 | 说明 |
